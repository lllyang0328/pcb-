# 个人简历
项目简介


2024.10-2025.02      基于UWB动态定位技术和智能交互系统的水上救援机器人      项目负责人

项目简介：
	为攻克传统水上救援中响应滞后、定位精度低及危险环境接近难等问题，我主导研发了一款集成了UWB高精度定位与边缘视觉感知技术的智能救援无人船。本项目以STM32微控制器为核心，通过融合UWB模块与K230边缘计算平台，构建了“精准定位-智能识别-自主决策-实时交互”的完整智能化救援流程，实现了对落水目标的快速发现、厘米级定位与自主抵近，显著提升了救援效率与安全性。

职责：
①以STM32为核心完成硬件底板PCB设计，主导从方案选型、原理图设计到PCB布局的全流程。数模/射频区域分割，解决高速数字电路、模拟采集电路及UWB射频电路之间的噪声干扰与信号完整性问题②负责UWB定位模块的硬件集成与接口设计，确保射频性能最优。设计多通道ADC采集电路，构建高实时性、低功耗的传感器数据采集系统③完成姿态、水位等多模态传感器的驱动电路设计、信号调理与硬件调试、并应用低通滤波优化数据精度。④负责外围功能单元的可调节电源模块设计与接口调试。
成果：国家级大学生创新创业训练计划项目
项目链接：国家级大学生创新训练计划平台(bjtu.edu.cn)

2025.02-2025.07                  智能生活垃圾分类一体机                        项目负责人 


项目简介：
	本项目致力于开发一款智能生活垃圾分类处理系统，STM32F4系列作为主控芯片统筹总体，作为高性能微控制器，兼具强大的数字信号处理能力、丰富连接性和实时控制功能。采用K230识别模块结合Yolov11图像算法，可对四类垃圾进行精准识别分类，分拣准确率达95%以上，多段履带传输系统配合机械滑台分拣，处理效率达每分钟60次以上。

职责：
	①主导完成控制主板设计，基于系统功耗、时序及接口需求规划电源与时钟架构，确保为核心芯片、传感器等提供稳定的运算与控制基础。②完成多层PCB设计，设计电源分配网络，电机驱动等大电流电路。③设计ADC采集与低通滤波处理确保数据精度，完成硬件系统级调试与验证。④完成总线架构设计，为传感器等外设完成I²C、SPI、UART等接口的电路布局，确保阻抗控制与等长布线，保障信号完整性与稳定通信。

成果：全国大学生嵌入式芯片与系统设计竞赛全国决赛三等奖，全国集成电路创新创业西北赛区二等奖
